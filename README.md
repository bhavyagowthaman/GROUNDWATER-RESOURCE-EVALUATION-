The frontend of this project runs on a local development server and can be accessed using http://localhost:3000. When the application is launched, users are directed to a login page where they must enter their username and password to securely access the system. This ensures that only authorized users can interact with the application features. The interface is designed to be user-friendly and responsive, providing a smooth experience for managing the system functionalities. During development, this local server helps in testing and debugging the frontend before deploying it to a live environment.
